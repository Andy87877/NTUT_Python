# 作業 05：Triangle

## 題目說明
請撰寫一個程式，判斷三個邊長是否能構成三角形，並計算其面積。

## 輸入說明
- 第一行輸入第一個邊長 a
- 第二行輸入第二個邊長 b
- 第三行輸入第三個邊長 c

## 輸出說明
- 如果三邊能構成三角形：
  - 第一行輸出 "Yes"
  - 第二行輸出三角形面積（保留小數點後兩位）
- 如果三邊不能構成三角形：
  - 輸出 "No"

## 範例輸入 1
```
3
4
5
```

## 範例輸出 1
```
Yes
6.00
```

## 範例輸入 2
```
1
2
3
```

## 範例輸出 2
```
No
```

## 範例輸入 3
```
5
5
5
```

## 範例輸出 3
```
Yes
10.83
```
