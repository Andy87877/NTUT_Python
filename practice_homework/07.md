# 作業 07：最佳資費選擇

## 題目說明
某電信公司提供兩種資費方案：
1. 方案A：月租費 100 元，每分鐘通話費 0.5 元
2. 方案B：月租費 200 元，每分鐘通話費 0.3 元

請撰寫一個程式，根據用戶每月通話時間，計算並推薦最適合的資費方案。

## 輸入說明
- 輸入一個整數，代表每月通話時間（分鐘）

## 輸出說明
- 第一行輸出方案A的總費用
- 第二行輸出方案B的總費用
- 第三行輸出推薦方案（A或B）
- 如果兩個方案費用相同，輸出 "都可以"

## 範例輸入 1
```
100
```

## 範例輸出 1
```
150
230
A
```

## 範例輸入 2
```
500
```

## 範例輸出 2
```
350
350
都可以
```

## 範例輸入 3
```
1000
```

## 範例輸出 3
```
600
500
B
```
